Final Project Proposal
Concept:
Teleportation via doors/entrances
I want to create a variety of rooms, each set
in a different environment. The main page will
have an assortment of doors on display. Each entrance is
a button that takes you to a new page with a different
environment when you click on it. I'll mainly use
Javascript and try to incorporate audio and
video backgrounds.
When you enter a room, there will be audio and a
video background which will make it seem like
you're actually in that space. To leave that room,
you click on a button that is hidden somewhere
in there and it takes you back to the main page or
somewhere else depending on what you click.
Some ideas I have for rooms:
- Campfire
- Outerspace
- Undersea
- Laser Tag
- Dinosaurs
- Concert
- Beach
